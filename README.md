# 🚀 Blinkit Sales Dashboard

## 📌 Overview

I'm excited to share my first Tableau Dashboard, built using **Tableau Public**.

This project analyzes Blinkit's sales data and presents key business insights through a clean and interactive dashboard. It is designed to help understand sales trends, outlet performance, and product distribution using different visualizations.
